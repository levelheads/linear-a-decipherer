# PART 1: ADMINISTRATIVE TABLETS

## 1.1 HT 13 (Hagia Triada) - Classic Wine Accounting

**Source**: GORILA, Younger transcriptions, ResearchGate
**Type**: Administrative tablet, commodity list
**Commodity**: Wine (VIN/B)

### Transliteration
```
Line 1:  KA-U-DE-TA
Line 2:  B(ina) . TE . RE-ZA         5½
Line 3:  TE-TU                       56
Line 4:  TE-KI                       28½
Line 5:  KU-DO-NI                    16½
Line 6:  DWO                         19
Line 7:  I-DU / NE-SI                5
Line 8:  KU-LO                       130½
```

### Analysis Notes
- **KA-U-DE-TA**: Heading position → likely toponym (cf. isle Kauda/Gavdos?)
- **B(ina)**: Wine logogram
- **TE-RE-ZA, TE-TU, TE-KI**: Names or place-names (note TE- prefix pattern)
- **KU-DO-NI**: Possibly Kydonia (Linear B: ku-do-ni-ja)?
- **DWO, I-DU, NE-SI**: Additional entries
- **KU-LO**: TOTAL formula (= ku-ro with l/r variation)
- **Sum verification**: 5.5 + 56 + 28.5 + 16.5 + 19 + 5 = **130.5** ✓ MATCHES

### Confirmed Anchors Used
- Level 2: ku-ro/ku-lo = "total" (positional verification ✓)
- Level 3: VIN/B = wine logogram (pictorial origin ✓)
- Level 4: Structural pattern (heading → entries → total ✓)

# PART 2: LIBATION FORMULAS (Religious Register)

## 2.1 Standard Formula Structure

**6-Element Pattern** (recurring across sites):
```
Element 1: a-ta-i-*301-wa-ja (variants: -wa-e, etc.)
Element 2: [PERSONAL NAME/LOCATION - varies per inscription]
Element 3: (j)a-sa-sa-ra-me (variants: a-sa-sa-ra-me)
Element 4: u-na-ka-na-si (variants: u-na-ru-ka-na-ti)
Element 5: i-pi-na-ma (variants: i-pi-ṇạ-ṃị-ṇạ)
Element 6: si-ru-te (+ additional elements)
```

**Interpretive Hypothesis**:
- Element 1: Dedicatory phrase (verbal root in -i-*301-)
- Element 2: Dedicant's name (wildcard position)
- Element 3: Direct object? "Gift, homage" (not necessarily deity name)
- Element 4: Verbal component "expressing hope for divine favor"
- Elements 5-6: Ritual/closing terms

# PART 3: KOBER METHOD ANALYSIS PREPARATION

## 3.1 Frequency Patterns Identified

### High-Frequency Signs (Corpus-Wide)
| Sign | AB# | Frequency | Positional Tendency |
|------|-----|-----------|---------------------|
| a | AB08 | 39.3% of vowels | Initial/medial/final |
| i | AB28 | 25.7% of vowels | Medial/final |
| u | AB10 | 18.1% of vowels | Medial |
| e | AB38 | 14.0% of vowels | Medial |
| o | AB61 | 2.9% of vowels | Rare (diagnostic!) |
| mu | AB23 | Very High | Medial/final |
| ti | AB37 | High | Final (inflection?) |

### Position Patterns
**Initial Position (High Frequency)**:
- a- (AB08): Very common prefix
- pa- (AB03): Toponyms (pa-i-to)
- ku- (AB81): Administrative (ku-ro)
- ja- (AB57): Religious (ja-sa-sa-ra-me)
- ta- : Libation formula (ta-na-ra-...)
- te- : HT 13 entries (te-re-za, te-tu, te-ki)

**Final Position (High Frequency)**:
- -ro (AB02): Administrative endings (ku-ro, ki-ro)
- -si (AB41): Verbal? (u-na-ka-na-si)
- -me (AB13): Formula element (ja-sa-sa-ra-me)
- -ja (AB57): Adjectival? (a-ta-i-*301-wa-ja)
- -te : Ritual? (si-ru-te)
- -ma : Ritual? (i-pi-na-ma)

## 3.2 Potential Kober Triplets

### Pattern: Same Root + Variable Endings

**Candidate 1: K-R Administrative Root**
```
ku-ro    (total)
ki-ro    (deficit)
ka-i-ro  (balance?)
```
- Same consonantal skeleton K-R
- Different vowel patterns → inflection or derivation?

**Candidate 2: -NA-KA-NA- Core**
```
u-na-ka-na-si   (IO Za 2)
u-na-ru-ka-na-ti (PK Za 11)
```
- Same core: -NA-KA-NA-
- Variable prefix (u-na- vs u-na-ru-)
- Variable suffix (-si vs -ti)
- Evidence of verbal conjugation?

**Candidate 3: TE- Prefix Names (HT 13)**
```
TE-RE-ZA
TE-TU
TE-KI
```
- Same prefix TE-
- Different roots
- Toponymic/ethnic designation?

## 3.3 Bridging Syllable Candidates

**Theory**: In CV syllabary, word-final consonant + suffix-initial vowel = bridging sign

**Example: -SI Endings**
```
u-na-ka-na-SI  (ends in -si)
If next morpheme starts with vowel, would see:
  -SI-a, -SI-e, -SI-i, etc.
```

**Search Target**: Find sequences where -si- appears word-internally after roots ending in -na.

## 3.4 Negative Evidence (Principle 5)

**ABSENT: Expected Greek Patterns**
- No -os, -on, -ōn nominal endings
- No Greek verbal morphology (-ō, -eis, -ei)
- ku-ro ≠ Linear B to-so
- Low /o/ frequency (2.9%) vs. Greek higher

**ABSENT: Semitic Triconsonantal Morphology**
- No root + pattern variations (K-T-B → kataba, kitāb)
- K-R root shows vowel variation but not Semitic pattern

**ABSENT: Extensive Case Inflection**
- Limited suffix variation vs. Linear B
- Possibly isolating or agglutinative
