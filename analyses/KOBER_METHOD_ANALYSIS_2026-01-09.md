# EXECUTIVE SUMMARY

This analysis applies Alice Kober's methodology to the Linear A corpus data compiled on 2026-01-09. The Kober Method prioritizes internal pattern analysis before linguistic interpretation.

**Key Findings**:
1. **K-R Root Paradigm**: ku-ro, ki-ro, (ka-i-ro) show vowel alternation suggesting morphological relationship
2. **Verbal Ending Candidates**: -SI/-TI alternation in libation formulas may indicate conjugation

4. **Register Differentiation**: Administrative and religious texts show distinct vocabulary patterns

# PART 1: THE KOBER METHOD APPLIED

## 1.1 Methodology

Alice Kober's approach to Linear B (which enabled Ventris's decipherment) consists of:

1. **Frequency analysis** - Track sign occurrence rates
2. **Positional analysis** - Map where signs appear (initial/medial/final)
3. **Triplet detection** - Find same roots with different endings
4. **Bridging syllables** - Identify where root consonant + suffix vowel combine
5. **Grid construction** - Arrange signs by shared consonant/vowel

This analysis applies steps 1-3 to current corpus data.

## 1.2 Core Principle

> "Anyone who took a theory about the language as their starting point was doomed to failure. The single route to the finishing line was to find the patterns and deep symmetries inside the primary texts." — Alice Kober

# PART 2: PATTERN FINDINGS

## 2.1 The K-R Root Pattern

### Data
```
ku-ro    50+ occurrences    List-final position    "total"
ki-ro    30+ occurrences    With deficit markers   "deficit/owed"
ka-i-ro  1 occurrence       ZA 4                   "balance"?
```

### Distributional Evidence
| Sequence | Position | Context | Semantic Domain |
|----------|----------|---------|-----------------|
| ku-ro | End of lists | After commodity entries | Summation |
| ki-ro | Within lists | With negative markers | Subtraction |
| ka-i-ro | Uncertain | ZA 4 context | Accounting? |

### Pattern Analysis

**Consonantal Skeleton**: All three share **K-R** structure

**Vowel Alternation**:
```
ku-ro:   u - o
ki-ro:   i - o
ka-i-ro: a - i - o
```

### Morphological Interpretation

**Option A: Derivational Relationship**
- Same root K-R with different prefixes/infixes
- ku- (totality), ki- (deficit), ka-i- (balance)
- Pattern: [prefix]-ro

**Option B: Ablaut/Vowel Gradation**
- Single root with vowel modification indicating meaning shift
- Similar to Semitic: K-L-L → kull "all", Arabic forms

**Option C: Unrelated Homophones**
- Coincidental similarity
- Different roots that happen to share K-R

### Test Required
Verify ka-i-ro across corpus:
- If appears consistently in "balance" contexts → Option A/B supported
- If only in ZA 4 → Possibly hapax, insufficient evidence

### Confidence Assessment
- ku-ro = "total": **HIGH** (50+ occurrences, consistent position)
- ki-ro = "deficit": **HIGH** (30+ occurrences, consistent context)
- ka-i-ro = "balance": **SPECULATIVE** (1 occurrence, needs verification)
- K-R as productive root: **POSSIBLE** (pattern suggestive, not proven)

## 2.2 Verbal Ending Candidates (-SI/-TI)

### Data
```
IO Za 2:  u-na-ka-na-SI
PK Za 11: u-na-ru-ka-na-TI
```

### Structural Comparison
| Component | IO Za 2 | PK Za 11 | Analysis |
|-----------|---------|----------|----------|
| Prefix | u- | u- | Identical |
| Element 1 | -na- | -na-ru- | Variation (+ru) |
| Core | -ka-na- | -ka-na- | Identical |
| Ending | -si | -ti | Alternation |

### Pattern Analysis

**Core Stability**: The sequence -KA-NA- is identical in both
**Frame Variation**: Different prefixes/infixes around the core
**Final Alternation**: -SI vs -TI

### Morphological Interpretation

**Option A: Verbal Conjugation**
```
-si = 3rd person singular?
-ti = 2nd person singular? / different tense?
```
- Supported by: Same syntactic position in parallel formulas
- Problem: Only 2 examples

**Option B: Dialectal Variation**
```
IO (Mount Iouktas) writes -si
PK (Palaikastro) writes -ti
```
- Supported by: Geographic separation of sites
- Problem: Would expect other dialectal differences too

**Option C: Scribal Variation**
- Same morpheme, different spelling conventions
- S/T confusion possible in syllabic script

### Additional -SI/-TI Data Needed
Search for:
- Other words ending in -si across corpus
- Other words ending in -ti across corpus
- Distribution patterns by site

### Confidence Assessment
- -SI/-TI represent endings: **PROBABLE** (positional evidence)
- -SI/-TI are grammatically distinct: **POSSIBLE** (too few examples)
- -SI/-TI indicate verbal conjugation: **SPECULATIVE** (hypothesis only)

## 2.3 The TE- Prefix Pattern

### Data (HT 13)
```
TE-RE-ZA    5½  (wine)
TE-TU       56  (wine)
TE-KI       28½ (wine)
```

### Pattern Analysis

**Common Feature**: All begin with TE-
**Context**: All are entries in wine accounting tablet
**Position**: All precede numerical quantities

### Interpretation Options

**Option A: Toponymic Prefix**
```
TE- = "from/of [place]"
TE-RE-ZA = "[product] from Reza"
TE-TU = "[product] from Tu"
TE-KI = "[product] from Ki"
```
- Parallel: Linear B uses toponymic adjectives for provenance

**Option B: Category Marker**
```
TE- = administrative category (e.g., "taxed", "received")
```
- All three are same type of entry in ledger

**Option C: Coincidence**
- Three different words that happen to share initial syllable
- No grammatical significance

### Test Required
- Search for TE- words across HT corpus
- Check if TE- appears in non-accounting contexts
- Compare frequency of TE- initial words to baseline

### Confidence Assessment
- TE- is recurring pattern: **CERTAIN** (visible in data)
- TE- is productive prefix: **POSSIBLE** (only 3 examples in 1 tablet)
- TE- indicates provenance: **SPECULATIVE** (untested)

## 2.4 Libation Formula Morphological Variation

### Complete Formula Structure
```
Element 1: a-ta-i-*301-wa-ja    (opening)
Element 2: [VARIABLE]           (personal name)
Element 3: (j)a-sa-sa-ra-me     (core term)
Element 4: u-na-ka-na-si        (verbal)
Element 5: i-pi-na-ma           (ritual)
Element 6: si-ru-te             (closing)
```

### Variant Forms Documented

| Element | Standard | Variant | Change Type |
|---------|----------|---------|-------------|
| 1 | -wa-ja | -wa-e | Final vowel |
| 3 | ja-sa-sa-ra-me | a-sa-sa-ra-me | j- loss |
| 4 | u-na-ka-na-si | u-na-ru-ka-na-ti | Internal + final |
| 5 | i-pi-na-ma | i-pi-ṇạ-ṃị-ṇạ | Extended form |

### Pattern Analysis

**J-/∅ Alternation**
```
JA-sa-sa-ra-me (IO Za 2)
A-sa-sa-ra-me  (PK Za 11)
```
- Suggests: j- is **prefix**, not part of root
- Root: A-SA-SA-RA-ME
- J- may be: article, demonstrative, or grammatical marker

**-JA/-E Alternation**
```
a-ta-i-*301-wa-JA
a-ta-i-*301-wa-E
```
- Both appear in same syntactic position (formula opening)
- Possible: Adjectival/agreement endings

### Morphological Conclusions

1. **Prefixation**: Linear A uses prefixes (j-, a-, u-)
2. **Suffixation**: Linear A uses suffixes (-ja, -e, -si, -ti, -me)
3. **Infixation**: Possible (-ru- insertion in u-na-ru-ka-na-ti)
4. **Reduplication**: Evident (SA-SA in a-sa-sa-ra-me)

### Language Type Implications
- **Agglutinative features**: Prefixes + suffixes
- **Limited inflection**: Compared to Greek
- **Reduplication**: Common in many language families
