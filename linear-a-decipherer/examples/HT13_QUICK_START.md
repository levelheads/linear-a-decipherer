## The Tablet

**HT 13** - Wine commodity list from Hagia Triada (c. 1450 BCE)

```
1. KA-U-DE-TA  VIN · TE ·
2. RE-ZA       5 ½
3. TE-TU       56
4. TE-KI       27 ½
5. KU-ZU-NI    18
6. DA-SI-*118  19
7. I-DU-NE-SI  5
8. KU-RO       130 ½
```

## The Six First Principles (Applied)

### ☑ P1: KOBER - Let Data Lead

**What we did**: Analyzed structure and patterns BEFORE making language assumptions.

**Pattern identified**:
- List structure: name + quantity (×6 entries)
- Final line: KU-RO + sum
- Sum verification: 5½ + 56 + 27½ + 18 + 19 + 5 = 130½ ✓

**Result**: Identified totaling function from structure alone, independent of language.

---

### ☑ P2: VENTRIS - Abandon When Contradicted

**What we did**: Tested multiple hypotheses and weighted evidence objectively.

**Example**: TE- prefix hypothesis (TE-TU, TE-KI both start with TE)
- Initial hypothesis: TE = category marker
- Evidence: Only 2 examples, high coincidence probability
- **Decision**: Flagged as SPECULATIVE, not claimed as proven

---

### ☑ P3: ANCHORS - Build from Certain Outward

**Anchors we used**:

| Anchor | Level | Confidence | Why |
|--------|-------|------------|-----|
| VIN = wine | 3 (logogram) | CERTAIN | Confirmed Linear B cognate |
| KU-RO = total | 2 (cognate + position) | PROBABLE | 37 corpus attestations in totaling position |
| TE = transaction sign? | 4 (structural) | POSSIBLE | Pattern unclear |
| Personal names | 5-6 | SPECULATIVE | No etymology |

**Result**: Confidence calibrated to highest anchor reached (Level 2-3).

---

### ☑ P4: MULTI-HYP - Test All Four Hypotheses

**KU-RO tested against all four linguistic hypotheses**:

| Hypothesis | Support | Best Evidence |
|------------|---------|---------------|
| **Semitic** | ★★★ STRONG | Akkadian *kullu "all" → phonetic + semantic match |
| Proto-Greek | ★★ MODERATE | Linear B parallel usage |
| Luwian | ★ WEAK | No clear cognate |
| Pre-Greek | ★ WEAK | No diagnostic features |

**Result**: Semitic hypothesis scores highest for KU-RO, but we acknowledge uncertainty.

---

### ☑ P5: NEGATIVE - Consider Absences

**What's NOT present in HT 13**:

❌ No toponyms (pa-i-to, etc.) → No Level 1 anchors available
❌ No Semitic triconsonantal patterns → Weak support for Semitic overall
❌ No Pre-Greek diagnostic clusters (-nth-, -ss-) → Weak support for Pre-Greek
❌ No clear Greek case endings (-os, -on, -oi) → Weak support for Greek

**Result**: Negative evidence constrains interpretation; prevents over-confident claims.

---

### ☑ P6: CORPUS - Verify Across All Occurrences

**KU-RO corpus verification**:
- Searched all 1,721 inscriptions
- Found 37 KU-RO attestations
- **100% appear in list-final/totaling position**
- ✅ Reading VERIFIED as corpus-consistent

**I-DU-NE-SI corpus check**:

- ⚠️ Confidence automatically downgraded to POSSIBLE

## The Workflow (Checklist)

Use this for analyzing any Linear A inscription:

### Phase 1: Preparation
- [ ] Run First Principles pre-flight checklist
- [ ] Acquire source (image, database, reference)
- [ ] Document metadata (site, date, context)

### Phase 2: Transliteration
- [ ] Identify each sign using AB numbers
- [ ] Check sign_list.md for phonetic values
- [ ] Note word boundaries and damage
- [ ] Separate logograms from syllabograms

### Phase 3: Pattern Analysis (KOBER)
- [ ] Analyze structure (list? religious text? label?)
- [ ] Check sign frequencies in corpus
- [ ] Note positional patterns (initial/medial/final)
- [ ] Identify repeated sequences

### Phase 4: Anchor Identification
- [ ] Level 1: Any toponyms? (pa-i-to, etc.)
- [ ] Level 2: Linear B cognates with position match?
- [ ] Level 3: Logograms?
- [ ] Level 4: Structural patterns?
- [ ] Level 5: Morphological patterns?
- [ ] Level 6: Lexical guesses?

### Phase 5: Multi-Hypothesis Testing
- [ ] Test against **Luwian** hypothesis
- [ ] Test against **Semitic** hypothesis
- [ ] Test against **Pre-Greek** hypothesis
- [ ] Test against **Proto-Greek** hypothesis
- [ ] Weight evidence objectively

### Phase 6: Cross-Corpus Verification
- [ ] Search corpus for all sequences
- [ ] Check for consistent usage
- [ ] Note hapax legomena (downgrade confidence)
- [ ] Verify proposed readings in all contexts

### Phase 7: Calibration & Output
- [ ] Calibrate confidence to highest anchor reached
- [ ] Document evidence chains
- [ ] Note uncertainties explicitly
- [ ] Run post-analysis First Principles check

## Common Pitfalls (Avoided)

❌ **Starting with language assumption**
- ✅ Instead: Analyze patterns first (P1: KOBER)

❌ **Ignoring contradicting evidence**
- ✅ Instead: Acknowledge and weight objectively (P2: VENTRIS)

❌ **Claiming certainty based on weak anchors**
- ✅ Instead: Calibrate to anchor level (P3: ANCHORS)

❌ **Testing only one linguistic hypothesis**
- ✅ Instead: Test all four systematically (P4: MULTI-HYP)

❌ **Focusing only on present features**
- ✅ Instead: Consider absences (P5: NEGATIVE)

❌ **Reading based on single occurrence**
- ✅ Instead: Verify across corpus (P6: CORPUS)
