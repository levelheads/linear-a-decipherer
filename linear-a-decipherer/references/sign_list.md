## Key Recurring Sequences

### Administrative
- `a-du` - Personal name or title
- `da-me` - Possibly "community/people"
- `pa-i-to` - Phaistos (toponym, confirmed via Linear B)
- `su-ki-ri-ta` - Toponym
- `se-to-i-ja` - Toponym

### Libation Formula (Religious)
Standard formula on stone vessels:
```
a-ta-i-*301-wa-ja | ja-sa-sa-ra-me | u-na-ka-na-si | i-pi-na-ma | si-ru-te
```

Components:
- `a-ta-i-*301-wa-ja` - Possibly "to the Lady" (dedicatory)
- `ja-sa-sa-ra-me` - Divine name? (Asasarame)
- `u-na-ka-na-si` - Verbal form?
- Variants exist across different vessels

## Numerical System

**Integers**: Decimal system
- | = 1
- — = 10
- ○ = 100
- ◎ = 1000

**Fractions**: Complex system with multiple symbols
- J, E, F, K, L, etc. represent various fractions
- System not fully understood

## Sign Identification Tips

1. **Orientation**: Linear A written left-to-right (rarely right-to-left or boustrophedon)
2. **Word dividers**: Dots or small vertical strokes between words
3. **Ligatures**: Some signs combine (complex signs)
4. **Damage**: Use brackets for uncertain readings
5. **Variants**: Same sign may have palaeographic variants across sites
