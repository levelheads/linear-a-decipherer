# Linear A Corpus

Overview of the Linear A inscription corpus and key reference materials.

> **⚠️ FIRST PRINCIPLE #6**: All readings must be verified across the ENTIRE corpus. A reading that works on one tablet but fails on others containing the same sequence is INVALID. See SKILL.md for full First Principles.

## Document Types

### Administrative Documents

**Clay Tablets**
- Primary accounting records
- List commodities, personnel, transactions
- Often fragmentary
- Found in palatial archives

**Roundels**
- Circular clay documents with seal impressions
- Function as receipts or tokens
- Unique to Linear A (not used in Linear B)

**Sealings**
- Nodules: Lumps attached to strings/containers
- Labels: Hanging tags
- Administrative authentication

### Non-Administrative Documents

**Stone Libation Tables**
- Religious offerings
- Contain "libation formula"
- More complete than tablets
- Found at peak sanctuaries

**Stone Vessels**
- Offering vessels with inscriptions
- Often contain divine names or dedications

**Metal Objects**
- Gold/silver pins with inscriptions
- Personal/votive objects

**Pottery**
- Inscribed ceramic vessels
- Potters' marks or ownership

## Standard Reference Corpus

### GORILA (Godart & Olivier, 1976-1985)

*Recueil des inscriptions en Linéaire A* - The authoritative print corpus.

| Volume | Contents |
|--------|----------|
| I | Tablets edited before 1970 |
| II | Nodules, sealings, roundels before 1970 |
| III | Documents 1975-1976 |
| IV | Other documents |
| V | Addenda, corrigenda, indices, sign charts |

**Format**: Black-and-white photo + drawing + transcriptions per document.

### Online Resources

**SigLA** (`https://sigla.phis.me`)
- Digital palaeographical database
- Searchable by sign, word, document
- Original drawings (not GORILA copies)
- Color-coded sign functions
- In development; administrative tablets focus

**John Younger's Website** (`http://people.ku.edu/~jyounger/LinearA/`)
- Phonetic transcriptions of all tablets
- Based on GORILA with corrections
- Transliteration using Linear B values
- Free access to normalized texts

## Key Inscriptions

### Administrative (Selected)

**HT 13** - Classic accounting tablet from Hagia Triada
- Multiple entries with commodities and numbers
- Contains `ku-ro` (total)

**HT 88** - Personnel list
- Names followed by quantities
- Transaction signs

**ZA 4** - Zakros tablet
- Well-preserved example
- Demonstrates regional variation

### Libation Formula

Standard religious formula appearing on multiple stone vessels:

```
Full form (composite from multiple sources):
a-ta-i-*301-wa-ja  ja-sa-sa-ra-me  u-na-ka-na-si  i-pi-na-ma  si-ru-te
```

**Key examples**:
- IO Za 2 (Mt. Iouktas)
- PS Za 2 (Psychro Cave)
- PK Za 11 (Palaikastro)
- KN Za 10 (Knossos)
- SY Za 4 (Kato Symi)

### Hagia Triada Administrative Archive

The HT tablets form the largest coherent archive:
- Commodity records (wine, oil, grain, figs)
- Personnel lists
- Possibly workshop inventories
- Dating: LM IB (c. 1500-1450 BCE)

## Naming Conventions

### Site Abbreviations
Two-letter codes from find site name.

### Document Numbers
Sequential within site: HT 1, HT 2, HT 3...

### Fragment Letters
Sub-fragments: HT 85a, HT 85b

### Document Type Suffixes
- No suffix: Tablets
- Za: Stone vessels/tables
- Zb: Other inscribed objects
- Wa: Roundels
- Wb: Sealings/nodules

**Example**: `IO Za 2` = Mt. Iouktas, stone vessel, #2

## Recent Discoveries (2024-2025)

### Knossos Ivory Scepter (2024)

**The most significant Linear A discovery in decades.**

| Attribute | Details |
|-----------|---------|
| Ring | **KN Zg 57** - Ceremonial/religious inscription |
| Handle | **KN Zg 58** - Administrative inscription |
| Location | Anetaki plot, southern precinct of Heraklion |
| Context | "Fetish Shrine" - semi-subterranean ritual deposit |
| Date | Neopalatial Period (1700-1450 BCE) |
| Sign count | 84 preserved + 35 partial = **~119 signs** (longest Linear A inscription ever) |

**Physical description**:
- Ivory ring + handle (possibly originally a single unit)
- Ring has inscription on all 4 faces
- Handle has separate inscription (different scribe/time)

**Ring inscription characteristics**:
- Refined, calligraphic style (resembles Cretan Hieroglyphs)
- NO numerals - suggests ceremonial/religious use
- Logograms: quadrupeds (12 animal signs), vases (10, with content indicators)
- Possibly records offerings for religious feast

**Handle inscription characteristics**:
- Standard Linear A administrative style
- INCLUDES numerals and fractions
- Different hand than ring inscription

**Publication**:
- Kanta, A., Nakassis, D., Palaima, T.G., Perna, M. (2025)
- "An archaeological and epigraphical overview of some inscriptions found in the Cult Center of the city of Knossos (Anetaki plot)"
- *Ariadne* Supplement 5 (KO-RO-NO-WE-SA Proceedings)
- DOI: https://doi.org/10.26248/ariadne.vi.1841
- PDF: https://ejournals.lib.uoc.gr/Ariadne/article/view/1841/1751

**Significance for decipherment**:
- Confirms dual register (administrative + religious) of Linear A
- Provides substantial new data for pattern analysis
- High-quality calligraphy allows better sign identification
- Ceremonial context helps interpret sign sequences
