## Executive Summary

When Greeks adapted Linear A to write Mycenaean Greek (~1450 BCE), they dropped **123 signs** representing phonemes Greek LACKED. This is a controlled experiment: we know Greek phonology; the deleted signs reveal what Minoan had that Greek didn't.

**Total unique signs**: 123
**Total attestations**: 874
**Top signs analyzed**: *301, *304, *188, *118, *86, *306, *21, *305

### Phonotactic Rules Emerging

1. **Initial-Only Phonemes**: *304, *188, *86 represent consonants that ONLY appear word-initially
   - Parallels: Semitic pharyngeals/emphatics, Luwian palatals

2. **Final-Only Phoneme**: *118 represents a consonant that appears word-FINALLY
   - Parallels: Word-final consonants in CVC languages
   - Greek dropped this because Linear B uses strict CV

3. **Classifier Signs**: *21F/*21M are grammatical markers, not phonemes
   - Implication: Minoan had grammatical gender marked by classifiers

4. **Logogram-Heavy**: *304, *188, *301 behave like commodities (standalone + numeral)

## Key Findings

### 1. Word-Final Consonant System (*118)
*118's 69% final position proves Linear A had **closed syllables** (CVC structure).
- Greeks couldn't represent this with CV-only Linear B
- Explains why *118 was dropped: no final consonant signs in Greek

### 2. Gender Classifier System (*21)
*21F/*21M prove Linear A had **grammatical classifiers**.
- Not phonemes but grammatical markers
- Greeks didn't need classifiers (mark gender morphologically)
- Suggests agglutinative or classifier language features

### 3. Khania Regional Phoneme (*86)
*86's 58% Khania concentration suggests possible **dialectal variation**.
- May represent sound specific to western Crete trade language
- Or metallurgical terminology (Khania = copper focus)

### 4. Semitic Administrative Layer Confirmed
Initial-position constraints of *301, *304, *188 match Semitic phonotactics.
- Pharyngeals and emphatics are strongly word-initial in Semitic
- Reinforces Contact Language model: Semitic administrative vocabulary layer
