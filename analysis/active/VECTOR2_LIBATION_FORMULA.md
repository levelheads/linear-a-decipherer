## Executive Summary

The libation formula has a FIXED structure appearing across 8+ sites over 200 years. The positions encode GRAMMATICAL SLOTS. By constraining by function (Position 1 = Verb, Position 4 = Divine name, etc.), we can decode backward from structure to meaning.

**Key Finding**: The complete formula appears on IOZa2 (Iouktas) with all 6 positions:

```
A-TA-I-*301-WA-JA | JA-DI-KI-TU | JA-SA-SA-RA-ME | U-NA-KA-NA-SI | I-PI-NA-MA | SI-RU-TE
    Position 1    |  Position 2  |   Position 3   |   Position 4   |  Position 5  | Position 6
```

## Position-by-Position Vocabulary Inventory

### Position 1: Main Verb/Opening Formula

**Pattern**: A-TA-I-*301-WA-JA and variants

| Form | Occurrences | Sites |
|------|-------------|-------|
| **A-TA-I-*301-WA-JA** | 11 | Iouktas (3), Syme (5), Kophinas (1), Palaikastro (1), Troullos (1) |
| A-TA-I-*301-WA-E | 1 | Palaikastro |
| TA-NA-I-*301-U-TI-NU | 1 | Iouktas |
| TA-NA-I-*301-TI | 1 | Psykhro |

**Morphological Analysis**:

| Segment | Analysis | Hypothesis |
|---------|----------|------------|
| A- | Prefix or prothetic vowel | Multiple |
| -TA- | Root consonant | TA = "to give"? (Luwian *dā-*) |
| -I- | Thematic vowel | Common in verbal forms |
| *301 | Unknown phoneme | /ħa/ or /kʲa/ (VECTOR 1) |
| -WA- | Particle | Luwian quotative -wa- |
| -JA | Suffix | Luwian adjectival -iya |

**Variant Analysis**:
- **-WA-JA** vs **-WA-E**: Dialectal variant or morphological difference?
- **TA-NA-** vs **A-TA-**: Different prefix or root variation?
- **-U-TI-NU**: Additional verbal/nominal ending?

**Proposed Function**: **VERB** (dedicatory action)
- "Offers/dedicates" or "libates to"
- Position 1 = verb-initial structure (VSO or V-initial)

**Confidence**: HIGH (consistent position 1; morphological structure)

### Position 3: Divine Name or Object

**Pattern**: JA-SA-SA-RA-ME / A-SA-SA-RA-ME

| Form | Occurrences | Sites |
|------|-------------|-------|

| A-SA-SA-RA-ME | 1 | Iouktas |

**Morphological Analysis**:

| Segment | Analysis | Hypothesis |
|---------|----------|------------|
| JA- / A- | Prefix alternation | May be optional; dialectal? |
| SA-SA- | Geminated syllable | Pre-Greek feature (cf. -ss- in toponyms) |
| -RA-ME | Suffix | Nominal ending or name element |

**Pre-Greek Phonological Features**:
- SA-SA gemination matches Pre-Greek pattern (like Parnass-os, Thalassa)
- -ME ending common in Cretan divine names

**Proposed Function**: **DIVINE NAME** (recipient deity)
- "Asasarame" or similar
- The deity to whom the offering is made

**Confidence**: PROBABLE (exclusive religious context; Pre-Greek phonology)

## Complete Formula Reconstruction

Based on position analysis:

### IOZa2 (Complete 6-Position Example)

```
Position 1: A-TA-I-*301-WA-JA     = "[Subject] offers/dedicates"
Position 2: JA-DI-KI-TU          = "[Name of dedicant]"
Position 3: JA-SA-SA-RA-ME       = "to Asasarame" (divine name)
Position 4: U-NA-KA-NA-SI        = "[epithet/verb]"
Position 5: I-PI-NA-MA           = "this offering"
Position 6: SI-RU-TE             = "through libation" / "from [this] pouring"
```

**Proposed Translation** (SPECULATIVE):

> "[The dedicant] JA-DI-KI-TU dedicates/offers to the goddess Asasarame, [epithet], this offering through libation."

## Verb Morphology Analysis

### A-TA-I-*301-WA-JA Paradigm

| Form | Difference | Possible Function |
|------|------------|-------------------|
| A-TA-I-*301-WA-**JA** | Standard | 3sg present? "offers" |
| A-TA-I-*301-WA-**E** | -E vs -JA | 3sg past? Or dialect? |
| TA-NA-I-*301-U-TI-NU | Different prefix + suffix | Different tense/aspect? |
| TA-NA-I-*301-TI | Reduced | Participle? |

**Luwian Verbal Parallels**:
- Luwian 3sg present: *-ti*
- Luwian 3sg past: *-ta*
- -WA-: quotative particle marking reported speech/oath

**Hypothesis**: A-TA-I-*301-WA-JA contains:
- Verbal root (TA = "give"?)
- Phoneme *301 (uncertain value)
- Luwian quotative -wa-
- Adjectival/agreement -ya

**Confidence**: POSSIBLE (morphological parallels; incomplete paradigm)

## Luwian Religious Formula Parallels

### Luwian Offering Inscriptions Structure

Luwian religious texts typically show:
1. Verb (dedicates/makes/gives)
2. Dedicant name
3. Deity name (often with epithet)
4. Object of offering
5. Circumstance/purpose

**Example** (Karatepe bilingual):
> "I am Azatiwada... I built this fortress... for Ba'al and for Kubaba..."

**Parallels with Linear A**:
- Verb-initial structure ✓
- Dedicant follows verb ✓
- Deity name with epithet ✓
- Object of dedication ✓

## Key Findings

### 1. Verb-Initial Structure CONFIRMED
Position 1 is consistently A-TA-I-*301-WA-JA → Verb-Subject-Object or Verb-initial formula.

### 2. Divine Name JA-SA-SA-RA-ME = Pre-Greek
- Geminated SA-SA matches Pre-Greek substrate phonology
- Exclusive religious context
- Possible goddess name "Asasarame"

### 3. Two Linguistic Layers in Linear A
| Register | Dominant Hypothesis | Key Terms |
|----------|---------------------|-----------|
| Administrative | Semitic (+47.4) | KU-RO, KI-RO, SA-RA₂ |
| Religious | Luwian/Pre-Greek (+14.5/+13.0) | A-TA-I-*301-WA-JA, JA-SA-SA-RA-ME |

### 4. -TE Suffix = Prepositional Function
SI-RU-**TE** in final position suggests -TE functions as preposition/case marker ("from/through").

### 5. Fixed Formula with Dialectal Variants
- Core formula consistent across 8+ sites over ~200 years
- Palaikastro shows variants (U-NA-RU-KA-NA-TI, I-PI-NA-MI-NA)
- Suggests standardized religious language with local adaptation
