## Methodology

Ventris broke Linear B by testing known toponyms against the grid. This is our equivalent: decompose personal names (especially from Khania) through Luwian morphological rules, testing whether names decompose cleanly under Luwian conventions.

**Corpus**: All multi-syllable words from 226 KH inscriptions (~92 unique forms), plus 111 corpus-wide name profiles.

## Key Finding: WI-SA-SA-NE as Hybrid

WI-SA-SA-NE is the most diagnostic name in the KH corpus:
- **WI-** = Luwian demonstrative *wi-* ("that one")
- **SA-SA** = Pre-Greek geminate root (cf. SA-SA-RA in libation formula)
- **-NE** = Luwian nominal suffix *-ni*

This is a **hybrid name**: Luwian grammar (prefix + suffix) wrapped around a Pre-Greek substrate root. This is exactly what we'd expect if Minoan Crete had a Pre-Greek substrate population that adopted Luwian-derived naming conventions through contact — the same pattern seen across Anatolia.

## Comparison with Luwian Divine Name Conventions

Luwian divine names follow patterns like *Tarhunza-* (storm god), *Runtiya-* (deer god), *Arma-* (moon god).

**Test**: Do any KH names contain elements that could be Luwian theophoric elements?

| Element | Luwian parallel | KH names | Result |
|---------|----------------|----------|--------|
| -MA suffix | *Arma-* related? | QA-NU-MA, QA-NU-MA | INCONCLUSIVE (could be nominal -ma) |
| -NE suffix | Luwian nominal | WI-SA-SA-NE, A-DI-NE, QA-NE, KI-SA-NE | CONSISTENT with naming patterns |
| WI- prefix | Demonstrative | WI-SA-SA-NE, WI-NA-DU | SUPPORTS Luwian contact |
| QA- prefix | Relative *kwa-* | QA-NU-MA, QA-NE, QA-*118, QA-PU, QA-TI-KI | SUPPORTS Luwian relative clause names |

**Conclusion**: KH names are consistent with Luwian naming conventions but do not contain recognizable Luwian divine name roots (Tarhunza-, Runtiya-, etc.). This suggests Luwian morphological influence WITHOUT theophoric content — the gods' names are different, but the grammar of naming is Luwian-derived.

## A- Prefix as Article/Demonstrative (Phase 4.3 Bold Bet)

The A- prefix appears on 15+ KH words:
- A-DU, A-DI-NE, A-DU-RE, A-MA-JA, A-KI-PI-E-TE, A-SE-RE-ZA, A-*301, A-RA, A-RA-U-DA, A-SI-*118, A-SI-KI-RA, A-TA, A-TA-*350, A-MI, A-ME-AROM

**Test**: Is A- an article/demonstrative (Luwian *a-* parallel)?
- A-DU vs DU: A-DU (7 occ, "contributor") vs DU (not independently attested as function word)
- A-TA vs TA: A-TA exists alongside TA as standalone → A- may be optional article

**Status**: SPECULATIVE. The high frequency of A-prefixed names is consistent with an article function, but we cannot yet distinguish between:
1. A- as productive article (like Luwian *a-*)
2. A- as integrated name element (not detachable)
3. A- as coincidental initial syllable
