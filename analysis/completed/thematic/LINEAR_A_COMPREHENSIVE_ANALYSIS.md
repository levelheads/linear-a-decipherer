## 3.2 Key Sequence Analysis: **ku-ro**

### 3.2.1 Distributional Evidence (Language-Neutral)

- **Frequency**: 50+ occurrences
- **Position**: **Always** end of commodity lists, before total numeral
- **Function**: Clear administrative "total" marker
- **Cross-corpus**: Verified HT, KH, ZA, PH sites

### 3.2.2 Hypothesis Testing

#### Hypothesis 1: LUWIAN/ANATOLIAN

**Proposed reading**: *kuru* or *kulla-*
**Phonological fit**: ✓ Luwian phonology allows CV.CV structure
**Lexical match**: ⚠ No clear Luwian word for "total/all"
**Morphological fit**: ? Luwian would expect **-ssa**, **-nti** inflections (not observed)

**Evidence strength**: **WEAK**

#### Hypothesis 2: SEMITIC (WEST SEMITIC/AKKADIAN)

**Proposed reading**: *kull* (with vowels inserted for CV syllabary)
**Consonantal skeleton**: K-R → K-L-L
**Lexical match**: ✓ **Akkadian *kalû/kullû*** = "all, totality, whole"
**Phonological fit**: ✓ Semitic root *KLL* widely attested
**Semantic fit**: ✓✓ **PERFECT** match with administrative "total" function

**Evidence chain**:
1. ku-ro appears with totals (positional)
2. Semitic *KLL* = "all/total" (lexical)
3. Administrative context matches Near Eastern accounting practices (cultural)
4. Other Semitic proposals: **ki-ro** ← *gry* "diminish", **su-pu** ← *'sp* "gather"

**Weaknesses**:
- Semitic uses consonantal script; Linear A has open syllables
- Could be **loan word** from trade, not genetic relationship
- No clear Semitic morphology elsewhere in corpus

**Evidence strength**: **PROBABLE** (lexical match + positional + cultural contact)

#### Hypothesis 3: PRE-GREEK SUBSTRATE

**Proposed reading**: Unknown substrate root
**Phonological fit**: ✓ Substrate allows any phonology
**Lexical match**: ✗ No attested Pre-Greek cognates for *kuro*
**Morphological fit**: ? Substrate morphology unknown

**Evidence strength**: **SPECULATIVE** (unfalsifiable = weak hypothesis)

#### Hypothesis 4: PROTO-GREEK

**Proposed reading**: **κύριος** (*kyrios*) "lord, master" → "complete/total"
**Phonological fit**: ✓ Greek allows *kyrios*
**Lexical match**: ⚠ **Semantic stretch** required ("lord" → "total")
**Morphological fit**: ✗ Greek would expect inflection for case; not observed
**Distributional problem**: ✗ Linear B uses **to-so** for "total", NOT cognate of **ku-ro**

**Evidence chain**:
1. **pa-i-to** = Phaistos suggests Greek presence
2. Continuity Linear A → Linear B suggests relationship
3. BUT: Linear B **to-so** ≠ Linear A **ku-ro** → **NEGATIVE EVIDENCE**

**Evidence strength**: **POSSIBLE** (but weaker than Semitic)

### 3.2.3 Synthesis: ku-ro

**Ranked hypotheses**:
1. **SEMITIC** (PROBABLE): Akkadian *kull* best lexical + semantic + positional fit
2. **PROTO-GREEK** (POSSIBLE): *kyrios* requires semantic stretch; negative evidence from Linear B
3. **PRE-GREEK** (SPECULATIVE): Unfalsifiable; no positive evidence
4. **LUWIAN** (WEAK): No clear cognate

**Methodological conclusion** (P4):
- Single-hypothesis support (Semitic strongest) → Max confidence: **PROBABLE**
- **ku-ro** likely **Semitic loan word** in Minoan administrative vocabulary

# PART 4: NEGATIVE EVIDENCE ANALYSIS

## 4.1 What Linear A Does NOT Show

Per **Principle 5 (Negative Evidence)**: Absence of patterns constrains hypotheses.

### 4.1.1 ABSENT: Clear Greek Cognates

**Observation**:
- **ONLY confirmed Greek word**: **pa-i-to** (toponym, likely substrate origin)
- **ku-ro** ≠ Linear B **to-so** (Greek "total")
- **ki-ro** = no Linear B equivalent
- No clear Greek case endings (**-os**, **-on**, **-ōn**)
- No clear Greek verbal forms (**-ei**, **-ousi**, **-etai**)

**Implication** (P5):
- **Minoan likely NOT Greek**
- Greek toponyms = **borrowed substrate names**, not proof of language identity
- **Negative evidence AGAINST Proto-Greek hypothesis**

### 4.1.2 ABSENT: Semitic Triconsonantal Morphology

**Observation**:
- Semitic languages have **root + pattern morphology** (K-T-B → kataba, kitāb, maktab)
- Linear A shows NO clear triconsonantal root patterns
- Open syllable structure (CV) obscures consonant clusters

**Implication**:
- **Minoan likely NOT genetically Semitic**
- Semitic vocabulary (**ku-ro**, **ki-ro**) likely **loan words from trade**
- **Negative evidence AGAINST genetic Semitic hypothesis**
- **SUPPORTS cultural contact model**

### 4.1.3 ABSENT: Extensive Inflectional Morphology

**Observation**:
- Linear B Greek shows **rich case inflection** (Kober's triplets)
- Linear A shows **LIMITED clear inflectional patterns**
- Fewer systematic morphological paradigms

**Implication**:
- Minoan may be **less inflected** than Indo-European languages
- Possible **isolating** or **agglutinative** language structure
- OR: Corpus too small to detect patterns (statistical limit)

### 4.1.4 ABSENT: Confirmed Luwian Grammatical Structures

**Observation**:
- Some Luwian particles proposed (**a-**, **-wa**, **-ja**)
- BUT: No clear Luwian case system (**-ssa**, **-nda** endings not confirmed)
- No clear Luwian verbal paradigms beyond speculative **-si**

**Implication**:
- **Luwian hypothesis has weak grammatical support**
- Particles could be **areal features** (Aegean Sprachbund) not genetic relationship

### 4.1.5 PRESENT: Strong Positional/Administrative Patterns

**Observation**:
- **Transaction words** (**ku-ro**, **ki-ro**, **te**) highly consistent
- Structural tablet format matches Near Eastern accounting
- Logogram + numeral system sophisticated

**Implication**:
- Minoan administrative practices borrowed from **Near East**?
- **Cultural contact model** supported
- Administrative vocabulary ≠ genetic linguistic relationship
