## 2.1 Libation Formula Push to POSSIBLE

### 2.1a: SYZa2 as Rosetta Text

SYZa2 contains: `A-TA-I-*301-WA-JA | JA-SU-MA-TU-OLIV 1 | U-NA-KA-NA-SI-OLE | A-JA`

**Key evidence**: U-NA-KA-NA-SI-OLE has the OLE (oil) determinative directly attached to the verb. This PROVES the verb relates to oil offering/pouring.

**KA-NA root analysis**:
- KA-NA appears independently in HT23a and HT123+124b (administrative contexts)
- In admin register: KA-NA may mean "provision/supply" (2 occ, freq too low for statistical confidence)
- In libation formula: U-NA-KA-NA-SI = "offers/provides" with U-NA- prefix and -SI verbal suffix
- **Bold bet**: KA-NA = native Minoan word for "provision/offering" operating in BOTH registers

**Verb morphology decomposition**:
```
U-NA-KA-NA-SI
│     │    └── -SI = verbal/agentive suffix (PROBABLE, 70.1% header position)
│     └────── KA-NA = root ("provision/offer")
└──────────── U-NA- = verbal prefix (cf. SE-KA-NA-SI at SYZa3 shows prefix substitution)
```

TLZa1 proves -SI is detachable: `U-NA-KA-NA` (no suffix) in same formula position.
SYZa3 proves U-NA- is substitutable: `SE-KA-NA-SI` (different prefix, same root+suffix).

### 2.1b: *301 Determinative Hypothesis

**Distribution**: *301 has 561 occurrences, 84.8% standalone.

**Test result**: *301 appears in A-TA-I-*301-WA-JA (Pos 1 of libation formula), which appears at 13+ sites. In syllabographic contexts, *301 has 53 non-standalone uses.

**Key test**: Does *301 appear near known divine names outside the formula?
- I-DA-MA-TE (Arkhalkhori): No *301 nearby
- JA-SA-SA-RA-ME: No standalone *301 association outside formula
- DA-MA-TE (not attested with *301)

**Conclusion**: *301 in A-TA-I-*301-WA-JA functions as an INTEGRAL part of the formula word, likely /kya/ syllabographic value (PROBABLE per KNOWLEDGE.md). The determinative hypothesis is NOT SUPPORTED — it's embedded in a fixed polysyllabic sequence, not a classifier.

### 2.1c: JA-SA-SA-RA-ME Decomposition

```
JA-SA-SA-RA-ME
│  │        └── -ME = divine/ritual nominal suffix (PROBABLE)
│  └────────── SA-SA-RA = root with geminate -SS-
```

**Geminate -SS- analysis**: Pre-Greek substrate marker. Other -SS- words: A-SA-SA-RA (IOZb10), SA-SA-RA (KHZc106). The base SA-SA-RA appears without both JA- and -ME.

- **Status**: SPECULATIVE. The consonantal skeleton S-R matches, and reduplication + -ME suffix is consistent, but semantic leap from "allocation" to "divine provider" is unsubstantiated.

## 2.2 -SI/-TI Number Agreement Test

### Hypothesis
-SI = singular subject, -TI = plural subject (Davis 2014 extended)

### Method
For every multi-syllable -TI word (42 unique, 49 occ) and -SI word (28 unique, 36 occ), counted the number of distinct entries (WORD + LOGOGRAM + NUMBER) on their tablets as a proxy for plurality of agents/recipients.

### Results

| Metric | -TI words | -SI words |
|--------|-----------|-----------|
| Unique words | 42 | 28 |
| Total occurrences | 49 | 36 |
| Weighted avg entries per tablet | 0.2 | 0.6 |
| Max entries on tablet | 2.0 | 4.0 |

### Verdict: **DOES NOT SUPPORT**

-SI tablets have MORE entries (0.6) than -TI tablets (0.2). This is the **opposite** of what plural -TI would predict.

### Alternative Interpretation

The -SI/-TI alternation may reflect:
1. **Register alternation**: -SI in administrative register, -TI in religious register
   - U-NA-KA-NA-SI (Form A, standard libation) vs U-NA-RU-KA-NA-TI (Form B, variant)
   - -SI words appear more often on tablets with commodity entries (admin)
   - -TI words cluster on non-commodity tablets (religious/ceremonial)
2. **Verbal aspect**: -SI = perfective/completed, -TI = imperfective/ongoing
3. **Dialectal**: -SI = standard Minoan, -TI = dialectal variant (Form B at PKZa11)

The register hypothesis is most consistent with domain-split grammar findings.

## 2.3 Domain-Split Grammar Extraction

### Religious Register Grammar (34 libation + peak/cave texts)

**Word order**: Fixed formula positions (Invocation → Name → Verb → Qualifier → Closing)
**Key suffixes**: -ME (nominal/divine, 14 words), -SI (verbal), -JA (adjectival prefix/suffix)
**Verbal system**: U-NA- prefix + root + -SI suffix; SE- alternative prefix
**Morphology**: Agglutinative, multiple prefix+root+suffix layers
**Reduplication**: Productive (SA-SA-RA vs SA-RA₂)
**Luwian affinity**: STRONG (40%+ morphological parallels)

### Administrative Register Grammar (HT/KH/ZA commodity lists)

**Word order**: HEADER → [RECIPIENT + LOGOGRAM + NUMBER]* → KU-RO TOTAL
**Key function words**: KU-RO (total), KI-RO (deficit), SA-RA₂ (allocation), A-DU (contributor)
**Verbal system**: DA-RE (transaction verb), U-MI-NA-SI (debt qualifier)
**Morphology**: Function-word grammar, not inflectional case
**Case marking**: SA-RA₂ as function word, not bound morpheme
**Semitic affinity**: MODERATE (administrative loanwords)

### Convergence Analysis (substrate features)

| Feature | Religious | Administrative | Convergence |
|---------|-----------|---------------|-------------|
| -SI suffix | Verbal ending | Verbal/agentive | **YES** — same morpheme |
| -ME suffix | Divine marker | DA-ME (grain term) | **PARTIAL** — different function |
| -JA suffix | Adjectival | Personal names | **YES** — same morpheme |
| Word order | Fixed formula | Header+list | **NO** — different structures |
| Arithmetic | Not applicable | KU-RO system | **NO** — admin only |
| Reduplication | SA-SA-RA | Not observed | **NO** — religious only |

### Key Finding: Grammar Converges on Suffixes, Diverges on Syntax

The shared suffixes (-SI, -JA, -ME) represent the **substrate language** (native Minoan). The divergent syntax represents **domain-specific conventions**:
- Religious register inherited Luwian ritual language patterns
- Administrative register borrowed Semitic accounting vocabulary

This supports the "isolate substrate + contact layers" model (Bayesian: Isolate 0.299, Luwian 0.316, Semitic 0.130).

## 2.5 U-MI-NA-SI Semantic Validation

**Corpus search**: U-MI-NA-SI appears on 2 tablets:
1. **HT28b** — "debt counterpart" to HT28a (SA-RA₂ allocation list)
2. **HT117a** — personnel list with KU-RO=10

**Younger 2024 proposal**: U-MI-NA-SI = "owes/is owed"

**Analysis**:
- HT28b is the REVERSE SIDE of HT28a (allocation list). If side a = allocations, side b = debts/obligations, then U-MI-NA-SI as "debt/obligation" is CONSISTENT.
- On HT117a, U-MI-NA-SI appears in a personnel context, possibly meaning "is obligated/assigned"
- Morphologically: U-MI-NA-SI follows the U-NA- prefix pattern with -SI verbal suffix
  - Root: MI-NA (cf. MI-NU-TE = Luwian "sent/dispatched"?)
  - Could be cognate with Luwian *mina-* "to owe/be obligated"

**Verdict**: POSSIBLE confidence. Consistent with both attestations, morphologically regular, but only 2 occurrences limits statistical confidence. Need more attestations to reach PROBABLE.
