## METHODOLOGY NOTES

### How Words Were Analyzed

1. **Frequency extraction** from pattern_report.json (1,071 unique words)
2. **Positional analysis** via Kober analyzer
3. **Multi-hypothesis testing** against Luwian, Semitic, Pre-Greek, Proto-Greek
4. **Cross-corpus verification** via regional_analysis.json
5. **PMI co-occurrence analysis** via contextual_analysis.json
6. **Slot grammar analysis** for commodity contexts

### First Principles Applied

All readings verified against FIRST_PRINCIPLES.md:
- P1 (Kober): Pattern before language assumption
- P2 (Ventris): Abandon contradicted hypotheses
- P3 (Anchors): Build from certain to speculative
- P4 (Multi-Hypothesis): Test all four frameworks
- P5 (Negative): Consider absent patterns
- P6 (Corpus): Verify cross-corpus consistency
