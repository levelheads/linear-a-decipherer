## Executive Summary

This analysis examines Linear A libation formulas - religious texts inscribed on stone offering tables found at peak sanctuaries across Crete. These represent a distinct textual register from administrative tablets.

**Key Findings**:
1. A **standard libation formula structure** exists with 5 core elements in fixed order
2. **JA-SA-SA-RA-ME** is confirmed as a divine name (7 occurrences, all peak sanctuaries)
3. Libation texts are **purely textual** - no numerals (unlike administrative tablets)
4. **Luwian influence** appears stronger in religious texts than administrative texts
5. Vocabulary shows **minimal overlap** with palace administrative register

## II. The Standard Libation Formula

### Composite Reconstruction

Based on cross-corpus analysis, the complete libation formula contains 5 core elements:

```
Position 1: A-TA-I-*301-WA-JA    (Opening invocation)
Position 2: JA-DI-KI-TU          (Secondary invocation/title)
Position 3: JA-SA-SA-RA-ME       (Divine name)
Position 4: U-NA-KA-NA-SI        (Divine epithet)
Position 5: I-PI-NA-MA SI-RU-TE  (Formula pair / closing)
```

## III. Core Vocabulary Analysis

### 1. A-TA-I-*301-WA-JA (Opening Invocation)

**Occurrences**: 11 confirmed
**Distribution**: All peak sanctuary contexts
**Variants**:
- A-TA-I-*301-WA-JA (standard) - IO, SY, TL, KO
- A-TA-I-*301-WA-E (with -E suffix) - PK Za 11
- JA-TA-I-*301-U-JA (JA- prefix variant) - AP Za 1
- TA-NA-I-*301-U-TI-NU (TA-NA- variant) - IO Za 6

**Structural Analysis**:
- Initial A-TA-I- : Consistent across variants
- *301 : Unknown sign, appears exclusively in religious context
- Final -WA-JA / -U-JA : Possible verbal ending

**Multi-Hypothesis Test**:

| Hypothesis | Interpretation | Support Level |
|------------|----------------|---------------|
| **Luwian** | *a-da-i-X-wa-ya* - contains Luwian particles -wa- (quotative) and -ya (connector) | **POSSIBLE** |
| Semitic | No clear cognates; -WA-JA not Semitic pattern | WEAK |
| Pre-Greek | Could be indigenous religious term | POSSIBLE |
| Proto-Greek | No Greek etymological parallels | WEAK |

**Assessment**: Luwian hypothesis supported by particles; Pre-Greek equally viable. Confidence: POSSIBLE.

### 2. JA-SA-SA-RA-ME (Divine Name)

**Occurrences**: 7 confirmed
**Distribution**: All occurrences at peak sanctuaries
**Variants**:
- JA-SA-SA-RA-ME (standard) - IO, PS, TL
- SA-SA-RA-ME (without initial JA-) - PK Za 11

**Structural Analysis**:
- JA- : Initial particle (common Minoan religious prefix)
- SA-SA : Geminated syllable (unique feature)
- -RA-ME : Divine name suffix (-ME appears in Minoan anthroponyms)

**Phonological Features**:
- Gemination SA-SA is highly distinctive
- Pattern *C-a-C-a-C-a-Me consistent across attestations
- High /a/ vowel frequency matches Pre-Greek substrate hypothesis

**Multi-Hypothesis Test**:

| Hypothesis | Interpretation | Support Level |
|------------|----------------|---------------|
| Pre-Greek | Indigenous divine name; gemination typical of Pre-Greek substrate | **PROBABLE** |
| Luwian | No clear Luwian divine name parallels | WEAK |
| Semitic | No Semitic divine name patterns | WEAK |
| Proto-Greek | No Greek divine name parallels | WEAK |

**Assessment**: Strong Pre-Greek candidate. Divine name likely indigenous to Minoan religion. Confidence: PROBABLE.

## IV. Structural Analysis: Libation vs. Administrative Texts

### Key Structural Differences

| Feature | Libation Formulas | Administrative Tablets |
|---------|-------------------|------------------------|
| **Numerals** | ABSENT (0%) | PRESENT (90%+) |
| **Logograms** | Rare (SY Za 2: OLIV, OLE only) | Common (GRA, VIN, OLE, etc.) |
| **KU-RO (total)** | ABSENT | Common (37 occurrences) |
| **KI-RO (deficit)** | ABSENT | Common (16 occurrences) |
| **SA-RA2** | ABSENT | Common (20 occurrences) |
| **Word dividers** | Present | Present |
| **Word order** | Fixed formula | Variable lists |
| **Average word length** | 4-6 syllables | 2-4 syllables |

### Vocabulary Non-Overlap

**Words EXCLUSIVE to libation texts**:
- A-TA-I-*301-WA-JA (and variants)
- JA-SA-SA-RA-ME
- U-NA-KA-NA-SI
- I-PI-NA-MA
- SI-RU-TE
- JA-DI-KI-TU

**Words EXCLUSIVE to administrative texts**:
- KU-RO
- KI-RO
- SA-RA2
- po-to-ku-ro
- Commodity logograms (routine use)

**Assessment**: The religious and administrative registers are **largely separate vocabularies**. This supports the "contact language" hypothesis with different linguistic layers for different social functions.

## V. Question Responses

### Q1: Is there a standard libation formula structure?

**YES**. A 5-position formula structure is confirmed:

```
1. Opening invocation (A-TA-I-*301-WA-JA)
2. Secondary invocation (JA-DI-KI-TU) [optional]
3. Divine name (JA-SA-SA-RA-ME)
4. Divine epithet (U-NA-KA-NA-SI)
5. Closing formula (I-PI-NA-MA + SI-RU-TE)
```

Confidence: HIGH

### Q2: How many divine names can we identify?

**One confirmed divine name**: JA-SA-SA-RA-ME

**Possible additional divine names** (less certain):
- A-TA-I-*301-WA-JA could be a divine name rather than invocation
- TA-NA-I-*301-U-TI-NU (IO Za 6) could be variant divine name

Confidence: PROBABLE (for JA-SA-SA-RA-ME); SPECULATIVE (for others)

### Q3: Does vocabulary differ from administrative texts?

**YES - SIGNIFICANTLY**.

The libation formula vocabulary shows virtually **zero overlap** with administrative vocabulary:
- No numerals
- No commodity logograms (except rare exceptions)
- No KU-RO, KI-RO, SA-RA2
- Unique religious terminology

This confirms **two distinct registers** in Linear A:
1. Administrative (palace economy)
2. Religious (peak sanctuary worship)

Confidence: HIGH

### Q4: Is Luwian influence stronger in religious texts?

**POSSIBLE - but not conclusively stronger**.

Evidence FOR Luwian in religious texts:
- -WA- particle in A-TA-I-*301-WA-JA (matches Luwian quotative -wa-)
- -JA suffix common (matches Luwian adjectival -iya)
- Formula structure recalls Anatolian invocation patterns

Evidence AGAINST exclusive Luwian:
- JA-SA-SA-RA-ME gemination not Luwian feature
- U-NA-KA-NA-SI not clearly Luwian
- Pre-Greek substrate equally viable for core vocabulary

**Comparative Assessment**:

| Register | Luwian Evidence | Semitic Evidence | Pre-Greek Evidence |
|----------|-----------------|------------------|---------------------|
| Administrative | WEAK | PROBABLE (KU-RO, SA-RA2) | POSSIBLE |
| Religious | POSSIBLE | WEAK | PROBABLE (JA-SA-SA-RA-ME) |

Confidence: POSSIBLE (Luwian stronger in religious texts, but Pre-Greek still dominant)

## VI. Negative Evidence Analysis (Principle 5)

### What is ABSENT from libation texts?

1. **Numerals**: Complete absence (0/14 inscriptions contain numbers)
   - Implication: Not accounting documents; purely devotional

2. **Commodity logograms**: Nearly absent (1 exception: SY Za 2)
   - Implication: Not recording offerings in administrative sense

3. **KU-RO and KI-RO**: Never appear
   - Implication: No totaling/deficit accounting in religious context

4. **Personal names in standard format**: Rare
   - Implication: Focus on divine names, not human dedicants

5. **Linear B cognate vocabulary**: Minimal
   - Implication: Religious register may be more conservative/indigenous

### What SHOULD appear if Proto-Greek?

If libation texts were Proto-Greek, we would expect:
- Greek divine name patterns (-eus, -on)
- Greek verbal endings
- Higher /o/ frequency

None of these are observed. **Negative evidence strengthens Pre-Greek hypothesis for religious register**.

## VIII. Discoveries and Novel Interpretations

### Discovery 1: Fixed Formula Structure

The libation formula has a **fixed 5-position structure** that is consistent across 8+ sites spanning 200 years. This suggests:
- Standardized religious practice across Crete
- Possible priestly/scribal tradition for ritual texts
- Central religious authority or shared tradition

Confidence: HIGH

### Discovery 2: Hybrid Inscription (SY Za 2)

SY Za 2 uniquely **blends administrative and religious elements**:
```
A-TA-I-*301-WA-JA | JA-SU-MA-TU OLIV 1 | U-NA-KA-NA-SI OLE | A-JA
```

This inscription contains:
- Religious formula elements (A-TA-I-*301-WA-JA, U-NA-KA-NA-SI)
- Commodity logograms (OLIV, OLE)
- A numeral (1)

**Interpretation**: This may be a **votive offering record** that bridges the administrative and religious registers.

Confidence: POSSIBLE (novel interpretation)

### Discovery 3: Variant Divine Name Forms

IO Za 6 presents a variant opening:
```
TA-NA-I-*301-U-TI-NU | I-NA-TA-I-ZU-DI-SI-KA | JA-SA-SA-RA-ME
```

This differs from standard A-TA-I-*301-WA-JA by:
- Starting with TA-NA- instead of A-TA-
- Ending with -U-TI-NU instead of -WA-JA

**Interpretation**: Either:
1. A different invocation to a different deity
2. A dialectal/temporal variant of the same formula
3. An extended form combining invocation + place name

Confidence: SPECULATIVE
