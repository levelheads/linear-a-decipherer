```

```

### Structural Analysis

| Line | Content | Type | Notes |
|------|---------|------|-------|
| 1 | I-NA-WA | HEADER | Possible title/deity |
| 2 | A-RI | NAME | Repeated (line 4) |
| 3 | I-ZU-RI-NI-TA | NAME | 5-syllable complex name |
| 4 | A-RI | NAME | Repeated from line 2 |
| 5 | I-DA-PA‚ÇÉ-I-SA-RI | NAME | 6-syllable complex; ends -RI |

### Key Observations

1. **NO numerals** - Unlike later tablets
2. **NO commodity logograms** - Pure name list
3. **NO KU-RO/KI-RO** - K-R vocabulary not present
4. **A-RI repeated** - Structural marker or common name
5. **Long names** - 5-6 syllables common in MMII

### Name Patterns

| Name | Syllables | Pattern | Notes |
|------|-----------|---------|-------|
| I-NA-WA | 3 | I-...-WA | -WA ending (Luwian?) |
| A-RI | 2 | A-RI | Repeated; possibly Semitic (* øari "lion"?) |
| I-ZU-RI-NI-TA | 5 | I-...-TA | -TA ending; -NI- infix |
| I-DA-PA‚ÇÉ-I-SA-RI | 6 | I-...-RI | Complex; ends -RI like A-RI |

### MMII Characteristics

Based on PH 6 and similar MMII tablets:

| Feature | MMII | MMIII+ |
|---------|------|--------|
| Numerals | Rare | Common |
| Commodity logograms | Absent | Present |
| KU-RO | Absent | Present (MMIII+) |
| Name length | 4-6 syllables | 2-4 syllables |
| Format | Name lists | Name + Quantity |

**Implication**: MMII represents proto-administrative phase; K-R vocabulary and numerical accounting developed in MMIII.

## Chronological Synthesis

### Vocabulary Development

| Period | K-R Vocabulary | Numerals | Commodities | Format |
|--------|---------------|----------|-------------|--------|
| **MMII** | ABSENT | Rare | ABSENT | Name lists |
| **MMIII** | **KU-RO present** | Common | Some (CAP) | Name + Quantity |
| **LMIB** | KU-RO + KI-RO | Universal | VIN/OLE/GRA/CYP | Full accounting |

### Key Finding

**KU-RO appears in MMIII** (PH(?)31a), pushing the chronology earlier than previously documented LMIB attestations at HT.

| Finding | Confidence | Implication |
|---------|------------|-------------|
| KU-RO in MMIII | **HIGH** | K-R vocabulary predates LMIB |
| KI-RO absent in MMIII | HIGH | Deficit tracking is later development |
| MMII lacks numerals | HIGH | Proto-administrative phase |

## Hypothesis Testing

### Semitic Analysis (K-R Vocabulary Origin)

If KU-RO = Semitic *kull* "all/total":

- MMIII presence suggests Semitic influence established by 1700-1600 BCE
- Consistent with Near Eastern trade contacts
- **Score**: HIGH fit

### Luwian Analysis (Name Patterns)

-JA endings (DI-PA-JA, etc.) suggest:
- Luwian patronymic suffix
- Pre-existing Anatolian population
- **Score**: MEDIUM fit for anthroponyms

### Pre-Greek Analysis

Long MMII names (I-ZU-RI-NI-TA, I-DA-PA‚ÇÉ-I-SA-RI):
- Do not match Semitic or Luwian patterns
- May represent indigenous substrate
- **Score**: MEDIUM fit for MMII layer
