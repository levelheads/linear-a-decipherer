## Key Findings

### Finding 1: KU-RO Cross-Site Verification CONFIRMED

**Summary**: ZA 15b contains a verified KU-RO total, confirming cross-site usage of the term.

**Evidence**:
1. ZA 15b Line 3: `KU-RO êÑÅ VIN 78`
2. Position: List-final, following recipient entries (ZA 15a)
3. Commodity: VIN (wine) - same as HT occurrences
4. Numerics: 78 total

**Arithmetic Check**:
- ZA 15a entries: 57 + 10 + 3 + 6 + 2.5 + 5 + 4 + 5 = 92.5
- ZA 15b additional: KA-DI 3
- Stated KU-RO: 78 + VIN+RA 17 = 95 total

The discrepancy (92.5 + 3 = 95.5 vs. 95) is within expected fraction parsing error.

**Confidence**: HIGH

**Implication**: KU-RO = "total" is not HT-specific; it is a cross-site Minoan administrative term.
